## 📌 About This Repository

This repository contains my learning journey and hands-on practice from the **Cognizant Digital Nurture 5.0 Deepskilling Program**.

It includes assignments, coding exercises, and implementations based on **Java Full Stack Development (FSE)** concepts.

The program focuses on backend, frontend, databases, testing, DevOps, cloud, and software development practices.

## 🎯 Objective

- Improve programming and problem-solving skills
- Learn full-stack application development
- Practice industry-based development concepts
- Build hands-on experience with modern technologies
